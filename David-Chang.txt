    1. What is Semantic HTML? 
    It is html that gives meaning to my markups.
    2. What is HTML used for? 
    It is used to create web pages.
    3. What is an attribute and where do we put it? 
    They are special flags placed inside of the tags that include information that we don't want to render on the page.
    4. What is the h1 tag used for? How many times should I use it on a page?
    It's the most important header on the page. You should use it once only on a page.
    5. Name two tags that have required attributes
    <img> and <a>
    6. What do we put in the head of our HTML document? 
    <header> + </header>, <h1> + </h1>, maybe a <nav> + </nav>
    7. What is an id? 
    It is an attribute added to an element, which then makes it possible for CSS to select that specific element more easily and accurately.
    8. What elements can I add an id to? 
    It can be added to any html element.
    9. How many times can I use the same id on a page? 
    You cannot have more than one element with the smae id in an HTML document.
    10. What is a class? 
    It is similar to an id, but allows for multiple html elements to share the same class.
    11. What elements can I add a class to? 
    It is part of the global attributes and can be added to any html element.
    12. How many times can I use the same class on a page? 
    As many times as needed within a document.
    13. How do I get my link to open in a new tab?
    You must add target="_blank" to your link/anchor tag. For example: <a href="https://www.google.com/" target="_blank">google.com</a>
    14. What is the alt attribute in the image tag used for? 
    It provides an alternate text for an image, if the image cannot be displayed.
    15. How do I reference an id?
    a href=#'nameofid'>
    16. What is the difference between a section and a div
    Section defines and/or references the content grouped inside, whereas the div does not tell you anything specific about a particular block of content.
    17. What is CSS used for? 
    It's used to add style and layout to web pages.
    18. How to we select an element? Example - every h2 on the page
    Element {}, for the example: h2{}
    19. What is the difference between a class and an id? - Give me an example of when I might use each one
    For id's you can only add to one element per document, whereas classes can be added to as many elements as needed. Ex: Within a grocery list, you might want to identify all categories (dairy, greens, meats, etc.) as part of the grocery list (class), but perhaps you want to single out one particular item and highlight it as a certain font color using id.
    20. How do we select classes in CSS?
    .classname{}
    21. How do we select a p element with a single class of “human””?
    p.human{}
    22. What is a parent child selector? When would this be useful? 
    It selects all elements that are a direct child of a specified element (the parent). It would be useful if you want to apply something to everything within a certain group, section, etc.
    23. How do you select all links within a div with the class of sidebar?
    .sidebar a{}
    24. What is a pseudo selector?
    Keyword added to a selector that specifics a special state of the select elements.
    25. What do we use the change the spacing between lines?
    {line-height:"";}
    26. What do we use to change the spacing between letters?
    {letter-spacing:"";}
    27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
    *{text-transform:uppercase;}, *{text-transform:lowercase;}, *{text-transform:capitalize;}
    28. How do I add a 1px border around my div that is dotted and black?
    div{style="border: dotted 1px black";}
    29. How do I select everything on the page? 
    *{}
    30. How do I write a comment in CSS?
    /*____________*/
    31. How do I find out what file I am in, when I am using the command line? 
    Type "pwd".
    32. Using the command line - how do I see a list of files/folders in my current folder?
    Type "ls".
    33. How do I remove a file via the command line? Why do I have to be careful with this? 
    rm "filename". Have to be careful because it won't prompt you before deleting the file.
    34. Why should I use version control? 
    Allows you to keep track of changes, collaborate with others, compare files, merge changes, etc.
    35. How often should I commit to github?
    Depends person to person but usually every 15-20 mins or when new features are added that are working and worth committing.
    36. What is the command we would use to push our repo up to github? 
    git push -u origin branch-name
    37. Walk me through Lambda's git flow. 
    Fork own version of repo, add TL as collaborator, copy the clone link, use CLI to clone the repo into the directory of choice, cd into new repo, create own branch name for repo, work on repo, add and commit changes, push changes back using git, and then intitiate a pull request using your own branch name, change base repo to forked repo, add TL to the reviewers, and then submit the pull request.
